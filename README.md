# Kubernetes CLI Learning Materials

## Overview
This repository contains educational materials for learning how to use the Kubernetes Command Line Interface (CLI).

## Prerequisites
Before running the materials, you need to have the following:
- A Kubernetes cluster running (can be local or in the cloud).
- kubectl installed on your machine.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/URK23CS1221/devex6.git
   cd devex6
   ```
2. Follow the setup instructions provided in the respective folders for each learning material.

## How to Use the Materials
- Study each guided lesson in sequence.
- Execute commands in your terminal as outlined in the materials.
- Troubleshoot any issues using the provided references.

## Running the Examples
1. Ensure that your terminal is configured with access to your Kubernetes cluster.
2. Use the provided YAML files to deploy resources by running:
   ```bash
   kubectl apply -f <file-name>.yaml
   ```
3. Monitor your resources with:
   ```bash
   kubectl get all
   ```

## Tips
- Familiarize yourself with basic kubectl commands to navigate through your Kubernetes installation.
- Join community forums and groups to get assistance and share knowledge.

## Conclusion
Using the resources in this repository, you will gain a solid understanding of how to interact with Kubernetes using the CLI. Happy learning!
